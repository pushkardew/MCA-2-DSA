## How to Run

Make sure Python 3 is installed on your system.

```bash
# Clone the repository
git clone https://github.com/leeladhar-bareth/DSA-Lab.git
cd DSA-Lab

# Run any program
python Unit1/01_insert_delete_list.py
python Unit3/15_merge_sort.py
python Unit5/26_n_queen_backtracking.py
```

No external libraries are required. All programs use Python's standard library only.
